# Flavor Code 图标重设计

日期：2026-07-19

## 背景

项目 `flavor-code` 是一个 Electron 桌面 AI 编程助手。当前图标（`assets/icon.svg`）为蓝底白色对话气泡笑脸，仅体现了"对话/助手"属性，未体现：
1. **Flavor**（风味/品尝）——名字的核心意象
2. **Code**（代码）——产品的领域属性

## 目标

重新设计图标，使之同时体现 flavor 和 code 双重含义，同时保持品牌延续性。新图标将同时用于：
- Electron 打包后的桌面应用图标（Windows 任务栏、桌面快捷方式、安装程序）
- Electron 窗口标题栏图标
- 未来可能的 favicon / CLI 品牌元素

## 设计方案

### 保留元素（品牌延续）
- 蓝色圆角方形背景（渐变 `#4a9fe8` → `#1979c9`，圆角半径 112/512）
- 白色对话气泡主体形状，保留右下角小尾巴尖角
- 24px 白色描边

### 调整元素
- **视觉居中**：将白色气泡整体向左上偏移约 8-10px，修正现有图标因右下角尾巴导致的视觉重心偏右下问题

### 新增/修改元素（脸部）
| 元素 | 原设计 | 新设计 |
|------|--------|--------|
| 眼睛 | 两个蓝色实心圆点（r=21） | 一对弯弯的花括号 `{ }`，深蓝色描边（`#1979c9`），左 `{` 右 `}`，作为笑眯眼 |
| 嘴巴 | 蓝色弧线微笑 | 半圆形张开的嘴，深蓝填充，呈"啊呜"俏皮嘴型 |
| 舌头 | 无 | 从嘴中伸出的椭圆形粉色舌头（珊瑚粉 `#ec4899` 系），舌尖微微朝下，呈 😛 俏皮吐舌状 |

### 语义映射
- **吐舌** → flavor（品尝、风味）→ AI "品味"代码
- **花括号 `{ }`** → code（JS/TS 等大括号语言最具代表性的符号）
- **对话气泡** → AI 助手、对话交互

### 技术落地
1. 重写 `assets/icon.svg` 为新设计（viewBox="0 0 512 512"）
2. 生成各尺寸图标资源：
   - `assets/icon-256.png` 更新
   - 新增 `assets/icon.ico`（Windows 多尺寸图标）
   - 新增 `assets/icon.icns`（macOS 图标集）
   - 或使用 `electron-builder` 自动从 SVG/PNG 转换
3. 在 `package.json` 的 `build` 配置中添加 `"icon": "assets/icon"` 字段
4. 在 `src/desktop/main.ts` 的 `BrowserWindow` 构造选项中添加 `icon` 属性
5. 验证窗口标题栏、任务栏、Alt-Tab 切换器中图标正确显示

## 不在范围内
- 不修改品牌色（蓝色渐变）
- 不修改 CLI 界面
- 不做深色/浅色模式变体（蓝底在两种模式下均有足够对比度）
